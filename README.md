# vue for cnode

> Vue项目cnodejs，使用了Vue，Vuex，Vue-router，Axios，Element-ui

## 编译安装

``` bash
# 安装依赖
npm install

# 开发和热加载在localhost:8833
npm run dev

# 编译成生成环境，可在服务器下直接预览
npm run build
```

## vue-cli

除vue-cli自带的一些，需要另外安装
```
npm i element-ui -S
npm i lodash -S
npm i timeago.js -S

```
