# vue for cnode

> Vue项目cnodejs，使用了Vue，Vuex，Vue-router，Axios，Element-ui

## 编译安装

``` bash
# 安装依赖
npm install

# 开发和热加载在localhost:8080
npm run dev

# 编译成生成环境，可在服务器下直接预览
npm run build
```
